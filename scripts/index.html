🔥 Firebase Real-Time Sync Connecting…
1
Create Project
Go to console.firebase.google.com → Add project → name it jad-chambers → disable Google Analytics → Create.
2
Add Web App
Inside the project → click Web icon (</>) → Register app → name it jad-chambers-web. Copy the firebaseConfig object shown.
3
Realtime Database
Left sidebar → Build → Realtime Database → Create Database → choose Asia-Southeast1 (closest to India) → Start in test mode.
4
Set Rules
Database → Rules tab → paste the rules shown below → Publish. This locks access to only your app's domain.
5
Paste Config
Paste each value from firebaseConfig into the fields below → click Connect to Firebase. All fields are saved locally.
📋 Firebase Security Rules — paste in Database → Rules tab
{ "rules": { "jadchambers": { ".read": true, ".write": true } } }
⚠ These are open rules for quick setup. Once working, restrict to your domain in Firebase Console → Authentication → Authorized domains, or add password auth later.
